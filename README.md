# EasyKick ⚽ — Football Pitch Booking System

A full-stack web application for booking football pitches online.

## Features
- User registration and login
- Browse and search available pitches
- Book pitches with date and time
- Admin panel to manage bookings, pitches and users
- Booking status system (pending / confirmed / cancelled)

## Tech Stack
- PHP, MySQL, HTML, CSS
- XAMPP (Apache + MySQL)
- Admin dashboard with full CRUD

## Setup
1. Import `easykick.sql` then `admin_migration.sql` into phpMyAdmin
2. Place files in `htdocs/easykick`
3. Visit `http://localhost/easykick`
4. Admin panel: `http://localhost/easykick/admin/login.php`
